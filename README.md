# Movie Recommendation System 
A content-based movie recommendation system built using Python and scikit-learn.

## Dataset
MovieLens (https://grouplens.org/datasets/movielens/)

## Technologies Used
- Python
- Pandas
- Scikit-learn
- TF-IDF Vectorizer
- Cosine Similarity

## Features
- Recommends similar movies based on genres
- Uses NLP techniques for similarity calculation
- Runs directly on Google Colab

## How to Run
1. Open the notebook in Google Colab
2. Run all cells
3. Call `recommend_movies("Movie Name")`
